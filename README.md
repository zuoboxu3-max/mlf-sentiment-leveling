# MLF：好感触レベリング・フィルターシステム（MLF Sentiment Leveling）

このリポジトリは、会話やテキストに対して「好感触度」をレベリング・フィルタリングする  
**MLF（Multi-Level Favorability / 好感触レベリング）システム** の研究・実装用 Google Apps Script プロジェクトです。

- Google スプレッドシートを入力インターフェースとして利用
- 発言・応対ログに対して「好感触レベル」を付与
- レベルに応じてフィルタリング・通知・可視化を行うことを目指した MVP / 研究プロジェクト
- clasp × GitHub でローカル開発 & バージョン管理

---

## 📁 プロジェクト構成

- GAS プロジェクト名：`MLF - 好感触レベリング・フィルター`
- このリポジトリ：`mlf-sentiment-leveling`
- 管理方法：Google Apps Script + clasp + Git + GitHub

`mlf-sentiment-leveling/` 配下の主なファイル：

- `コード.js` — MLFロジック本体（好感触レベリング・フィルター処理）
- `appsscript.json` — GAS プロジェクト設定
- `.clasp.json` — clasp 設定（スクリプトIDとの紐付け）

---

##  好感触レベリングのイメージ

現在は「好感触レベリングのロジック」を研究中です。  
想定している指標の例：

- 丁寧さ（敬語・配慮表現の有無）
- ポジティブ語・ネガティブ語のバランス
- 相手への共感・ねぎらいフレーズ
- クッション言葉（「もしよければ」「お手数ですが」など）

レベル設計案（例）：

- **Lv1**：最低限の受け答え
- **Lv2**：丁寧だが機械的
- **Lv3**：配慮・共感が含まれる
- **Lv4**：相手の状況に合わせた一歩踏み込んだ対応
- **Lv5**：感動レベルのホスピタリティ
